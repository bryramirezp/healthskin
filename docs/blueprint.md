# **App Name**: Healskin Landing Page

## Core Features:

- Hero Section: Visually appealing hero section with a clear value proposition of HEALSKIN.
- Problem/Solution Section: Illustrates the problem of traditional wound care and presents HEALSKIN as the innovative solution.
- Key Benefits Section: Highlights the 5 key benefits (accelerated recovery, rapid regeneration, less invasive, reduced pain, fewer infections).
- Science and Innovation Section: Showcases the biotechnology behind HEALSKIN.
- Origin Story Section: Introduces the founders and the university project origins of HEALSKIN.
- Contact Form Modal: A non-functional contact form placeholder to allow users to express interest.
- Static Instagram Feed: Placeholder Instagram feed using static images from picsum.photos.

## Style Guidelines:

- Primary color: Corporate Blue (#2E90FA), reflecting innovation and trustworthiness.
- Background color: Light desaturated blue (#E5F1FF), for a clean and modern feel.
- Accent color: Lime Green (#A7F3D0), to highlight the biotechnology aspect and recovery process.
- Body and headline font: 'Inter', sans-serif for modern, clear and accessible text.
- Lucide Icons for simple, modern icons related to wound care, recovery and biotechnology.
- Clean, responsive layout using TailwindCSS grid and flexbox. Each section should be distinct with clear headings and concise text. Ensure all the text is highly SEO optimized.
- Subtle fade-in animations on scroll for elements to enhance user experience.
